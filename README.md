# Planetary Energy Balance Lab

This repository contains a vivid single-file interactive climate simulation.

## What it teaches

- How incoming solar radiation, albedo, and greenhouse gases influence net radiative flux.
- Why warming can continue due to ocean heat uptake (thermal inertia).
- How scenario choices (preindustrial, present day, high emissions, mitigation) change risk indicators.

## Run locally

Open `index.html` in a browser.

Or start a local static server:

```bash
python3 -m http.server 8000
```

Then browse to <http://localhost:8000>.
